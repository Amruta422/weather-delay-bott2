# AI Log

## Prompt 1

Create a JavaScript Node.js approach for reading a local `orders.json` file, fetching weather for each city concurrently with `Promise.all`, and updating orders based on weather conditions.

## Prompt 2

Suggest clean error-handling logic for a weather-processing script where one invalid city should be logged but must not stop the remaining valid city requests.

## Prompt 3

Draft a simple personalized "weather-aware apology" function in JavaScript for delayed orders caused by rain, snow, or extreme weather.

## How AI assistance was used

- To refine the concurrent processing approach.
- To shape resilient error-handling behavior.
- To draft the apology-message function structure.

## Final implementation ownership

The final code was reviewed, organized, and adapted into this repository for the assignment requirements.
